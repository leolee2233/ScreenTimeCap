# ScreenTimeCap

ScreenTimeCap 是一个基于 Python 的轻量定时自动截屏工具。程序按照设定的时间间隔自动捕获屏幕，并以规范的格式自动命名并保存到指定文件夹中。

---

## 🚀 功能特点

* **定时自动截屏**：支持自定义截屏时间间隔（单位：秒）。
* **规范命名**：截屏文件采用 `序号-时间戳` 格式命名（例如 `001-20260612140000.png`），自动补零，极易归档和排序。
* **退出**：支持 `Ctrl + C` 安全终止程序，不抛出异常。

---

## 🛠️ 快速开始
使用screenshot_timer.py，需要安装pillow。可以在代码中修截屏保存地址。

或者直接使用screenshot_timer.exe，截屏将保存在D:\Screenshots中。
